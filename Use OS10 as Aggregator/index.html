Use OS10 as Aggregator

In this test case the goal is to create a simple load balancer using a reverse LAG port. The idea is to have one input port which is then mirrored to a logical LAG port and at the other end of the LAG port is a number of security sensors.

Physical Configuration

2 10Gb/s fiber SFPs one each connected to two separate VMs. These will serve as our traffic generators. A third 10Gb/s SFP is plugged into a third VM and this will serve as our receiver.

Test Scenario

I played a PCAP made up entirely of HTTP from port 1 and PCAP made entirely of DNS from port 2 and then listened on the VM attached to port 3. Confirmed, all traffic showed up on port 3 as expected.
